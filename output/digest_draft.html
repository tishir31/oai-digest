Product Launches & Updates

  • OpenAI Shuts Down Sora Video App and API, Citing Unsustainable Economics (2026-03-24) — OpenAI is shutting down its Sora video app on April 26 and API on September 24 due to high compute expenses, with daily inference costs estimated at $15 million against just $2.1 million in lifetime revenue. The core research team will refocus on world simulation for robotics. [TechCrunch]
  • OpenAI Pivots ChatGPT Shopping from Instant Checkout to Product Discovery via Agentic Commerce Protocol (2026-03-24) — Following struggled merchant onboarding and low sales conversion, OpenAI is replacing its Instant Checkout feature with an Agentic Commerce Protocol focused on dynamic product discovery. The pivot launches with integration from major retailers including Target, Home Depot, and Walmart in-app. [CNBC]
  • OpenAI Shelves Adult Mode 'Citron' Chatbot Indefinitely After Employee and Investor Pushback (2026-03-26) — Plans for an adult-oriented chatbot mode were indefinitely abandoned due to combined pushback from employees and investors concerning brand safety and moderation capabilities. [TechCrunch]
  • ChatGPT Launches Library Feature for Persistent File Management Across Conversations (2026-03-23) — A new persistent Library function allows users to automatically save, index, and reuse files generated or uploaded across distinct conversations. [OpenAI Help Center]
  • ChatGPT Launches Optional Location Sharing for Local Recommendations (2026-03-26) — OpenAI introduced an opt-in device GPS sharing capability for iOS and Web to fuel local search queries, directly positioning ChatGPT to compete for local search intent. [PPC Land]
  • OpenAI Adds Plugins Directory to Codex for Packaged Workflows with Skills, Apps, and MCP Servers (2026-03-26) — OpenAI launched a plugins directory for Codex that bundles reusable workflows combining skills, app integrations, and MCP servers. Users can browse and install plugins through the Codex app or CLI, formalizing a distribution ecosystem for OpenAI's coding agent product. [OpenAI Developers]
